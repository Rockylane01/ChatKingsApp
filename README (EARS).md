

User Requirements (EARS)
Ubiquitous
1. The ChatKings system shall allow users to participate in group chats with persistent
point totals.
2. The ChatKings system shall assign the King of the Chat role to the user with the highest
point total.
3. The ChatKings system shall allow chats to persist for a configurable duration up to one
year.
Event Driven
4. When a correct answer is selected by a minority of participants, the ChatKings system
shall award bonus points.
5. When the King of the Chat creates a prediction question, the ChatKings system shall
require the King to place a wager on that question.
6. When a prediction question is resolved, the ChatKings system shall award points to
users with correct answers.
7. When the King of the Chat is inactive beyond a configured time limit, the ChatKings
system shall revoke the King role and adjust point totals accordingly.
State Driven
8. While a user holds the King of the Chat role, the ChatKings system shall restrict question
creation to that user.
9. While a prediction question is active, the ChatKings system shall display the total points
wagered without revealing distribution by answer.
